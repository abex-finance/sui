pub fn generate_struct_layouts(&self) -> Registry

Generate layout schemas for all types declared by this package, as well as all struct types passed into entry functions declared by modules in this package (either directly or by reference). These layout schemas can be consumed by clients (e.g., the TypeScript SDK) to enable BCS serialization/deserialization of the package’s objects, tx arguments, and events.
