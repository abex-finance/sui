pub fn try_new(
    registry: &Registry,
    num_buckets: usize
) -> Result<Self, PrometheusSpanError>

Create a new layer, injecting latencies into the given registry. The num_buckets controls how many buckets thus how much memory and time series one uses up in Prometheus (and in the application). 10 is probably a minimum.
